极墨 · 简介

极墨 是一款移动端 Markdown 编辑器，纸墨相发，意在笔先。

  • 支持 CommonMark + GFM 语法
  • GitHub 仓库即数据源
  • 电脑 Typora + 手机极墨，无缝衔接
# 示例代码
const jimo = new Editor()
jimo.write('Hello 墨')

💡 设计理念

纸墨相发 — 宣纸暖底 + 浓墨黑字，长时间写作不累眼

朱砂点睛 — 红色只用于保存按钮和关键操作，如印章落款

去 UI 化 — 编辑器沉浸时隐藏工具栏，只留文字与纸面

暗如砚台 — 暗色模式如墨砚，红色键如朱砂印
